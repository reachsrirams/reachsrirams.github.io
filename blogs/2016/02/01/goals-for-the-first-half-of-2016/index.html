I am late by a month in sharing my blogging and learning goals for the first half of 2016. Let’s start with a review of the second half of 2015.

Status of 2015 second half goals

  1. YouTube video related to OpenStack – making a video is not easy but I believe I have nailed down the topic for the video.
    • Score: 10⁄10
    • Status: I published a YouTube video on OpenStack Neutron Ports.
  2. Build and blog about a small OpenStack GUI related idea – this could be 1-2 blogs
    • Score: 0/10
    • Status: I have done some ground work but no blogs, so no score
  3. Release the installation scripts for Kilo on Github
    • Score: 3⁄10
    • Status: I have the working code in my GitHub account – check out the dev/kilo branch. It works well but requires some finishing touches.
  4. Write a management related blog on LinkedIn – something new that I want to attempt
    • Score: 0/10
    • Status: Again I have written up a skeleton blog but nothing official to show
  5. Write 1 blog in the WILT series
    • Score: 10⁄10
    • Status: Published the ip vs ipconfig WILT blog.

If you look at the goals alone, it is clearly a mixed bag. I should have stretched a bit more completed more of the goals I set for myself. But considering the fact that I co-authored a book on OpenStack and spoke at the OpenStack Summit in Tokyo, I am happy with the accomplishments.

Goals for the first half of 2016

I am working on an idea for my second book so the goals for the first half will be somewhat conservative. OpenStack will continue to be the dominant theme.

  1. Build and blog about a small OpenStack GUI related idea – this could be 1-2 blogs
  2. Release the installation scripts for _Liberty_ release of OpenStack on GitHub
  3. Write a management related blog on LinkedIn

Given that only 5 months are left, I believe the goals above are reasonable taking into account personal and professional workload. I have carried forward these goals for a while and I am determined to complete them this time, so that I can pursue some fresh ideas for the second half of 2016
